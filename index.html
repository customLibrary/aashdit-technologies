Aashdit Component Library

Introduction

Aashdit Technologies Component Library is a modern, lightweight CSS framework that provides beautifully designed UI components for building responsive web applications. With a focus on simplicity and customization, our library offers a comprehensive set of components that are easy to use and integrate into any project.

Beautiful Design

Clean, modern components with attention to detail

Fully Responsive

Works perfectly on all device sizes

Lightweight

Minimal footprint for faster loading

Easy Customization

Variables and utilities for easy theming

Flexbox Utilities

1. Display

.d-flex, .d-inline-flex, .d-flex-inherit 

<div class="d-flex">...</div>

2. Flex Direction

.flex-row, .flex-column, .flex-row-reverse, .flex-column-reverse 

<div class="d-flex flex-column">...</div>

3. Wrapping

.flex-wrap, .flex-nowrap, .flex-wrap-reverse 

<div class="d-flex flex-wrap">...</div>

4. Justify Content

.justify-start, .justify-center, .justify-end, .justify-between, .justify-around, .justify-evenly 

<div class="d-flex justify-between">...</div>

5. Align Items

.items-start, .items-center, .items-end, .items-baseline, .items-stretch 

<div class="d-flex items-center">...</div>

6. Align Content

.content-start, .content-center, .content-end, .content-between, .content-around, .content-stretch 

<div class="d-flex flex-wrap content-between">...</div>

7. Align Self

.self-start, .self-center, .self-end, .self-stretch 

<div class="d-flex">
  <div class="self-start">A</div>
  <div class="self-center">B</div>
  <div class="self-end">C</div>
</div>

8. Gap

.gap-1 → 4px, .gap-2 → 8px, .gap-3 → 12px, etc. 

<div class="d-flex gap-3">...</div>

9. Row & Column Gap

.gx-* for column gap, .gy-* for row gap. 

<div class="d-flex flex-wrap gx-3 gy-2">...</div>

10. Flex Sizing

.flex-0, .flex-1, .flex-auto, .flex-grow-*, .flex-shrink-* 

<div class="d-flex">
  <div class="flex-1">Grow</div>
  <div class="flex-0">Fixed</div>
</div>

11. Flex Basis

.basis-25, .basis-50, .basis-75, etc. 

<div class="d-flex">
  <div class="basis-25">25%</div>
  <div class="basis-50">50%</div>
  <div class="basis-25">25%</div>
</div>

12. Order

.order-first, .order-last, .order-1, .order-2, etc. 

<div class="d-flex">
  <div class="order-3">1</div>
  <div class="order-1">2</div>
  <div class="order-2">3</div>
</div>
